The Aggregation Framework is analytic tools in mongoDB allowing you to run various reports or analysis on one or more collections. Aggregation modes include 1. aggregation pipeline 2. map-reduce 3. single purpose aggregation operations The pipeline gets passed down through the collection and put into different stages do produce the output. stage operators have different functions. $match can find the specifics of what you want to match. so saying you want the data from a certain year you would write { $match: { founded_year: 2015 }} $sort gives you a specific order of how you want your data displayed like -1 would give you alphabetically starting a z for our last lab the syntax would be db.recipes.aggregate([ { $match: { founded_year: 2015 }}]) most common stage operators are
  • $group
  • $project
  • $match
  • $limit
  • $skip
  • $unwind
  • $sort
